# プロジェクト概要

- React SPA 19.x / TypeScript 6.x
- npm/npx を使用（pnpm不可）

# プロジェクトの目的

- C#エキスパート、Next.js初中級の僕がネイティブなReact学習用
- 直近で受注した案件で必要な技術を習得用
  - Zustand
  - WebSocket
  - Leaflet

# コーディング規約

- 変数・関数: camelCase / コンポーネント: PascalCase
- 1ファイル1コンポーネント
- 学習用プロジェクトなので、本来不要だったり冗長なコメントを許容する
